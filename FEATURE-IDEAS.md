# Token Ledger — 功能扩展建议

基于当前 v0.8.0 代码分析，以下是可以增加的功能方向，按优先级和实现难度分组。

---

## 一、数据导出与分享

### 1. CSV / JSON 导出
- **描述**：将当前筛选范围内的用量数据导出为 CSV 或 JSON 文件，方便在 Excel、Google Sheets 或其他工具中二次分析。
- **难度**：低

### 2. 报表截图 / PNG 导出
- **描述**：一键将当前页面（总览、模型分析）截图为 PNG 图片，方便分享到社交平台或文档中。
- **难度**：中

---

## 二、增强分析维度

### 3. 项目维度统计
- **描述**：按 `.claude/projects/` 下的子目录名（即项目名）分组统计 token 消耗，展示哪些项目最"烧钱"。
- **难度**：中

### 4. 周环比 / 月环比对比
- **描述**：在总览页展示本周 vs 上周、本月 vs 上月的 token 消耗对比，用涨跌箭头和百分比表示变化趋势。
- **难度**：低

### 5. 小时级用量热力图
- **描述**：展示一天中各小时段的使用热力图（类似 GitHub contribution graph），帮助用户了解自己的使用习惯。
- **难度**：中

### 6. 成本预测
- **描述**：基于近 N 天的平均日消耗，预测未来 7 天 / 30 天的 token 消耗和成本。
- **难度**：低

### 7. 会话级别分析
- **描述**：按 `session_id` 或文件名聚合，展示单次会话的 token 消耗、持续时间和调用轮数，帮助识别"重型会话"。
- **难度**：中高

---

## 三、用户体验优化

### 8. 全局搜索与模型筛选
- **描述**：在模型列表页添加搜索框和筛选器，支持按模型名、provider 快速过滤。
- **难度**：低

### 9. 键盘快捷键
- **描述**：支持 `1/2/3/4` 切换页面、`R` 刷新数据、`F` 聚焦日期筛选等常用快捷键。
- **难度**：低

### 10. 系统托盘最小化
- **描述**：支持最小化到系统托盘，可设置开机自启，后台静默扫描。
- **难度**：中

### 11. 数据变更自动刷新
- **描述**：监听 Claude JSONL 目录和 MiMo 数据库的文件变化，自动刷新数据，无需手动点击"刷新数据"。
- **难度**：中

### 12. 页面内模型用量下钻
- **描述**：在模型卡片上点击后展开该模型的每日趋势图（利用已有的 `modelDaily` 数据），而非跳转到另一个页面。
- **难度**：低中

---

## 四、数据管理

### 13. 数据归档与清理
- **描述**：提供界面让用户查看本地数据量、归档旧日志或清理超过指定天数的历史数据。
- **难度**：中

### 14. 多数据源目录配置
- **描述**：允许用户自定义 Claude JSONL 扫描路径和 MiMo 数据库路径，支持多个目录。
- **难度**：中

---

## 五、通知与提醒

### 15. 成本阈值告警
- **描述**：用户可设置每日 / 每月成本阈值，超过时弹出系统通知提醒。
- **难度**：低中

### 16. 每日用量摘要通知
- **描述**：可选开启每日定时通知，汇总前一天的 token 消耗和成本。
- **难度**：中

---

## 六、开发者体验

### 17. 多语言支持（i18n）
- **描述**：当前界面全为中文，可增加英文等语言选项，扩大用户群体。
- **难度**：中

### 18. 自动更新
- **描述**：集成 Tauri 的 updater 插件，支持应用内检查更新并自动下载安装。
- **难度**：中
